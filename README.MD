# 🧮 Feet Measurement Equality System

> A clean, scalable **Quantity Measurement System** built using Java, designed to demonstrate real-world software design evolution through multiple use cases (UC1 → UC8).

---

## 🚀 Overview

This project showcases how a measurement system can be incrementally designed and improved using:

- Clean Code Practices
- Object-Oriented Design (OOP)
- Scalable Architecture Principles
- Git-based Feature Development Workflow

It focuses on **accurate unit comparison, conversion, and arithmetic operations** across different measurement units like Feet, Inch, Yard, and Centimeter.

---

## 🎯 Goals

✔ Compare measurements across different units  
✔ Enable seamless unit conversion  
✔ Support arithmetic operations (Addition of lengths)  
✔ Eliminate redundancy using **DRY Principle**  
✔ Design for scalability and future extension
